# Frontend Updates 2

## FE-1 Flask interface and presentation layer

- Added a responsive Flask UI with grouped farm input fields, a result page, and a polished earth-tone visual theme.
- Added dynamic region-to-state filtering in the browser for cleaner location selection.
- Added a JSON prediction endpoint so the recommendation flow can be used from the UI or programmatically.

## Manual steps

- Start the web app with `python app.py`.
- Open `http://127.0.0.1:5000/` in a browser to submit field conditions and view the crop and fertilizer recommendation.
